# StadiumScout (https://stadium-scout-8c177070e034.herokuapp.com)

StadiumScout is a web application that displays information about stadiums around the world. It was developed using HTML, CSS, Javascript in the front end, Node.js in the backend, and MongoDB for the database. It connects to Cloudinary API for image storing and MapBox API for map display. Users have the ability to register/login, create, edit, and delete stadiums as well as viewing them on a map, adding pictures, and leaving reviews.

To run this app locally: 
1) Run 'npm install' in root folder to generate node_modules folder
2) Run 'node app.js' in the root folder and make sure mongo back end service is running
